# Generator v2 Training Dataset

This version uses GRPO (reinforcement learning) and generates diverse prompts during training.

**Full dataset**: https://huggingface.co/datasets/yoavf/svg-animal-illustrations

The file `dataset_example.jsonl` in this directory contains a small sample for reference.

For training, download the full dataset from HuggingFace and place it as `dataset.jsonl` in this directory.
